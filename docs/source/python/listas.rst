****************
Listas
****************

Para acceder a los elementos de las listas se usan procedimientos de:

- _indexing_ (para acceder a un único elemento)
- _slicing_ (para acceder a una porción de la lista)



.. code:: python
    >>> x = [1, 2, 3, 4, 5, 6, 7, 8, 9]
    >>> x[0]
    1
